# Tools

Random tools I made and find helpful.

## Usage

> [!TIP]
> See "All Tools" below for internal links to each tool's README for details and
> execution instructions.

You can use the `tool_select.sh` script to find, select, and
execute your tool of choice.

```bash
git clone https://github.com/JoshSLawrence/tools
cd tools
chmod +x ./tool_select.sh
./tool_select.sh
```
